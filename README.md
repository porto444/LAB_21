Lab 21 - Capteurs embarques Android
Application Android (Java) qui permet de:

afficher la liste des capteurs disponibles
consulter les caracteristiques techniques d'un capteur
visualiser les mesures en temps reel sous forme de graphe
faire une reconnaissance simple d'activite a partir des capteurs de mouvement
Prerequis
Android Studio
Android SDK installe
JDK 17 (Android Studio JBR convient)
Lancer
Ouvrir le dossier du projet dans Android Studio, attendre la synchro Gradle, puis Run.

Tests rapides
Onglet "Capteurs": liste + recherche.
Ouvrir un capteur: ecran details.
"Graphe": choisir accelerometre/gyroscope et observer les courbes.
"Activite": poser le tel, marcher, courir (estimation simple).
